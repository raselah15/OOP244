In Workshop 4, I learned how to implement different types of operator overloading in a class. This workshop helped me understand how member operators such as +=, -=, <<, >>, prefix and postfix increment and decrement work with objects. I also practiced implementing type conversion operators like operator bool, operator int, operator double, and operator const char*. These operators helped me understand how objects can behave like built-in types in C++.

Another important concept I learned was how to manage different object states such as valid, new, and invalid states. I understood why checking the validity of an object before performing operations is important to avoid logical errors. Using the subscript operator also helped me understand how to safely access and modify characters in a C-string.

One difficulty I faced was handling file structure and Git workflow while working between my local machine, GitHub, and Matrix. However, this improved my confidence in using terminal commands, committing changes, and pushing code properly. Overall, this workshop strengthened my understanding of object-oriented design, operator overloading, and safe programming practices in C++.
